1. 아래와 같이 hello.sh 을 작성하고
```
#!/bin/sh
echo "Hello Word!!"
```
 
2. 실행권한을 부여한 후
```
chmod 700 hello.sh
```

3. 아래와 같이 실행
```
./hello.sh
```
